Hands-on Activity

For this hands-on activity, your task is to build an accessible HTML page. Start with the provided HTML Template below, and apply the principles of semantic HTML, proper heading structure, and ARIA landmarks that you've learned. Focus on making the content easily consumable by assistive technologies.
