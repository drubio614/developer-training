Best Practices for Accessible Lists

Using proper list elements is essential for structure and navigation, especially for screen reader users.

When to Use Ordered and Unordered Lists

Unordered Lists (<ul>): Use for a list of items where the order does not matter. Screen readers will typically announce this as a "list" with a certain number of items, and each item is often read with "bullet" or a similar indicator. Examples include shopping lists, feature lists, or navigation links.

Ordered Lists (<ol>): Use for a list of items where the sequence is important. Screen readers will announce this as a "numbered list" and read out the number of each item (e.g., "1 of 5," "2 of 5"). This is crucial for things like step-by-step instructions, rankings, or legal clauses.

Using the correct list element provides essential context that visual styling alone cannot. It allows users of assistive technologies to understand the structure of the content and navigate through it efficiently.

Examples

Good Examples

These examples use list elements correctly based on the context of the content.

Pancake Ingredients:

  • Flour
  • Eggs
  • Milk
  • Sugar
  • Baking Powder

How to Make Pancakes:

  1. Mix dry ingredients in a bowl.
  2. Add wet ingredients and whisk until smooth.
  3. Pour batter onto a hot, greased pan.
  4. Cook until bubbles form, then flip.
  5. Serve warm.
<!-- Correct: Using <ul> because the order of ingredients doesn't matter -->
<p><strong>Pancake Ingredients:</strong></p>
<ul>
    <li>Flour</li>
    <li>Eggs</li>
    <li>Milk</li>
</ul>

<!-- Correct: Using <ol> because the sequence of steps is critical -->
<p><strong>How to Make Pancakes:</strong></p>
<ol>
    <li>Mix dry ingredients in a bowl.</li>
    <li>Add wet ingredients and whisk until smooth.</li>
    <li>Cook until bubbles form, then flip.</li>
</ol>

Bad Examples

These examples fail to use proper semantic list elements, creating barriers for users of assistive technology.

Features (Incorrectly made with paragraphs):

- Fast performance

- Secure browsing

- Cross-platform sync

<!-- Incorrect: Simulating a list with <p> tags and hyphens. -->
<!-- A screen reader will just read these as three separate paragraphs, -->
<!-- not as a structured list of three items. -->
<p>- Fast performance</p>
<p>- Secure browsing</p>
<p>- Cross-platform sync</p>
